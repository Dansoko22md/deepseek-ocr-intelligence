# Frontend for deepseek-ocr-intelligence

This frontend will allow users to upload images, view extracted/cleaned text, and see structured JSON results from the backend API.

## Features (to implement)
- Image upload (single & batch)
- Display OCR text
- Display cleaned/structured JSON
- Export results (CSV/JSON)
- Dashboard for batch/document management

## Quickstart (to be implemented)
- Install dependencies
- Start the development server
- Connect to backend API at http://localhost:8000
